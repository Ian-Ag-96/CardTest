Prerequisites
	- Ensure postgres at least v 17 is installed.
	- Create a database called card_test after logging in.
	- Open command line from the folder with the project and run the command psql -U username -d database_name -h hostname -p port_number -f card-test.sql
	- Modify application.properties with your local url, username, and password
	- Ensure maven is installed
	- Run mvn clean install
	- Ensure docker is installed
	- Ensure these ports are free: 8081 for the application and 5431 for the postgres database.
	- Run the command docker-compose up --build
